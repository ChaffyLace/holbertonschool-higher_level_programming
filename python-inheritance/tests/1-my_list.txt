The MyList Class
================

On teste d'abord si la classe existe et si elle hérite bien de list.

    >>> MyList = __import__('1-my_list').MyList
    >>> list_class = MyList()
    >>> isinstance(list_class, list)
    True

Test basique (celui du main.py) :
On vérifie que l'affichage est trié, mais que la liste originale reste inchangée.

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test avec des nombres négatifs :

    >>> my_list_neg = MyList()
    >>> my_list_neg.append(-10)
    >>> my_list_neg.append(2)
    >>> my_list_neg.append(0)
    >>> my_list_neg.print_sorted()
    [-10, 0, 2]

Test avec une liste vide :

    >>> my_list_empty = MyList()
    >>> my_list_empty.print_sorted()
    []
